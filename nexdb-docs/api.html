API Reference

Complete reference guide for NexDB developer APIs, covering database creation, collection CRUD methods, indexing, migrations, and remote TCP sockets.

Viewing examples in:

Open Database

Initiate the database instance. Opens an existing database or creates a new one at the path. Replays the WAL automatically on startup for crash recovery.

open-database 
// Signature: pub async fn open(path: impl AsRef) -> NexDbResult
let db = NexDb::open("app.nex").await?;
# Signature: NexDb(db_path: str)
db = NexDb('app.nex')
db.connect() # Initiates subprocess connection
// Signature: constructor(dbPath: string)
const db = new NexDb('app.nex');
await db.connect(); // Starts subprocess connection

Collection Management

Manually create, drop, check, and list collection tables in the database.

collections-api 
// Explicitly create collection
db.create_collection("users").await?;

// Drop collection
db.drop_collection("users").await?;

// Check collection existence
let exists = db.has_collection("users").await; // returns bool

// List all collection names
let list = db.list_collections().await; // returns Vec<String>
# Create collection
db.create_collection('users')

# Drop collection
db.drop_collection('users')

# List collections
cols = db.list_collections() # returns list of strings
// Create collection
await db.createCollection('users');

// Drop collection
await db.dropCollection('users');

// List collections
const cols = await db.listCollections(); // returns Array

WAL Control & Checkpoint

Checkpoint commits all in-memory changes into database snapshots and flushes the Write-Ahead Log.

checkpoint-api 
// Forces WAL flush to disk
db.flush().await?;

// WAL Checkpoint: Snapshots in-memory collections and truncates WAL log file
db.checkpoint().await?;
# Flush WAL changes to disk
db.flush()
// Flush changes to disk
await db.flush();

Insert Document

Insert a JSON document under a custom collection table with an explicit string ID key.

insert-api 
// Signature: pub async fn insert(&self, collection: &str, id: &str, doc: Document) -> NexDbResult<()>
let doc = Document::from_value(serde_json::json!({
    "name": "Jane", 
    "role": "admin"
}));
db.insert("users", "u2", doc).await?;
# Signature: insert(collection: str, id: str, document: dict)
db.insert('users', 'u2', {
    'name': 'Jane',
    'role': 'admin'
})
// Signature: insert(collection: string, id: string, doc: object)
await db.insert('users', 'u2', {
  name: 'Jane',
  role: 'admin'
});

Insert with Auto-Generated ID

Insert document and generate a new UUIDv4 key automatically.

insert-auto-id 
// Signature: pub async fn insert_auto_id(&self, collection: &str, doc: Document) -> NexDbResult
let doc = Document::from_value(serde_json::json!({"item": "pen", "price": 1.5}));
let generated_id = db.insert_auto_id("products", doc).await?; // returns String
# Signature: insert_auto_id(collection: str, document: dict) -> str
generated_id = db.insert_auto_id('products', {'item': 'pen', 'price': 1.5})
// Signature: insertAutoId(collection: string, doc: object) -> Promise
const generatedId = await db.insertAutoId('products', { item: 'pen', price: 1.5 });

Get Document

Retrieve document contents matching the unique ID.

get-api 
// Signature: pub async fn get(&self, collection: &str, id: &str) -> NexDbResult
let doc = db.get("users", "u2").await?; // returns Document
# Signature: get(collection: str, id: str) -> dict
doc = db.get('users', 'u2') # returns dict
// Signature: get(collection: string, id: string) -> Promise
